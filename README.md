## Описание программы

**Weather Diary** — это графическое приложение для ведения дневника погоды. Программа позволяет добавлять записи о температуре, осадках, фильтровать их по дате и температуре, а также сохранять все данные в JSON-файл.

**Что умеет программа:**
- Добавлять записи с датой, температурой, описанием и отметкой об осадках
- Фильтровать записи по дате (за конкретный день)
- Фильтровать записи по температуре (выше заданного значения)
- Сохранять данные в JSON-файл и загружать из него
- Удалять записи (по одной или все сразу)
- Экспортировать данные в текстовый файл
- Показывать статистику (средняя температура, количество дождливых дней)
